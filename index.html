Configura el acceso a Google Sheets · Solo una vez

Paso 1 — Crear proyecto Cloud

Habilitar Google Sheets API

  1. Abre console.cloud.google.com
  2. Selector de proyectos (arriba) → Nuevo proyecto → nombre ReporteOp → Crear
  3. Menú ☰ → APIs y servicios → Biblioteca
  4. Busca Google Sheets API → Habilitar
Paso 2 — Crear credencial OAuth

Obtener tu Client ID

  1. Menú ☰ → APIs y servicios → Credenciales
  2. + Crear credenciales → ID de cliente OAuth
  3. Pantalla de consentimiento: tipo Externo, llena nombre y correo soporte → Guardar
  4. Tipo de aplicación: Aplicación web
  5. En "Orígenes de JavaScript autorizados" agrega estas URLs:
    http://localhost
    file://
    Si lo alojas en un servidor, también agrega ese dominio.
  6. Clic Crear — copia el ID de cliente
